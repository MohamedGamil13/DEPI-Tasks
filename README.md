## 🚀 Getting Started

To run any task locally:

1. Clone the repository
   ```bash
   git clone https://github.com/MohamedGamil13/<repo-name>.git
   ```

2. Navigate to the task folder
   ```bash
   cd task_01
   ```

3. Install dependencies
   ```bash
   flutter pub get
   ```

4. Run the app
   ```bash
   flutter run
   ```
